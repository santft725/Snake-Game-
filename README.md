🐍 Snake Game

A classic Snake Game built using Python. Control the snake, eat food, and grow as long as possible without crashing!

🎮 How It Works

Move the snake around the screen
Eat the food to grow longer
Avoid hitting the walls or yourself
Try to achieve the highest score possible

🚀 Features

Smooth snake movement
Score tracking
Increasing difficulty as you grow
Clean and simple game logic

🧠 Lessons Learned

Object-Oriented Programming (OOP) using classes and self
Implementing and managing multiple objects
List slicing and data handling
Game loop logic and collision detection

🛠️ Tech Used
Python
Turtle
